# Contributing to iLoveAudio

Thanks for your interest in contributing.

## Development Setup

```bash
git clone https://github.com/AgrimLabs/iloveaudioediting-webapp.git
cd iloveaudioediting-webapp
npm install
npm run dev
```

## Making Changes

1. Create a branch: `git checkout -b feature/your-feature`
2. Make your changes
3. Run `npm run build` and `npm run lint`
4. Commit with clear messages
5. Open a Pull Request

## Code Style

- TypeScript for type safety
- Tailwind for styling
- Keep components small and focused

## Reporting Issues

Use the [GitHub Issues](https://github.com/AgrimLabs/iloveaudioediting-webapp/issues) with a clear description and steps to reproduce.
